# realtime-sync-orchestrator

Sync status across distributed systems. Terminal UI built with [Bubble Tea](https://github.com/charmbracelet/bubbletea).

## Run

```bash
go run .
```

## Build

```bash
make build
./bin/realtime-sync-orchestrator
```

## Controls

- `j/k`: navigate
- `Enter`: select/details
- `q`: quit
